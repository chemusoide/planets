    Cómo viaja una petición por el sistema

    Aquí se explica el recorrido navegador → Nginx → FastAPI → SQLite → respuesta JSON.

    1

    Navegador

    Todavía no se ha lanzado ninguna petición.

    2

    Proxy web

    Nginx recibirá la ruta /api y la enviará al backend interno.

    3

    Backend FastAPI

    La API procesará la ruta y pedirá datos al servicio de planetas.

    4

    Persistencia SQLite

    La base de datos leerá los planetas guardados en el volumen persistente.
